PROCESO DE SINCRONIZACIÓN:
1.Cree la voz en ElevenLabs, al ser un plan gratuito tuve que utilizar una de las voces de stock, elegi la mas acertada para mi personaje.
Por otro lado, tome una imagen de Ursula (mi personaje) y la subi a HeyGen para crear mi avatar.

2. Usando SUNO IA, cree una musica ambiental acorde con el manual de marca y estetica que estuve trabajando en modulos anteriores.
Una musica instrumental, tranquila, con tono alegre y acogedora.

PROMP PARA CANCIÓN
"Chill art-café ambience with upbeat, inspiring afrobeats energy at a relaxed mid-tempo. 4. Una vez con la voz lista fui a HeyGen y lo sume a mi nuevo avatar para crear el video. Lastimosamente HeyGen no te permite siquiera descargar un video, por lo que tuve que grabar la pantalla para poder descargarlo.

5. Finalmente, cuando tuve el video animado de mi avatar con el audio, lo uni con la musica de fondo en clipchamp para obtener el resultado final.
